# Install Rust development kit on your Ubuntu System
Information based on: https://www.rust-lang.org/tools/install

## Using rustup 
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

rustc --version
```
## Update rust 
```bash
rustup update
```

## Uninstall rust 
```bash
rustup self uninstall
```
